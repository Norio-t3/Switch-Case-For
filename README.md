# Switch-Case-For

"for": Inicia o loop para repetição.

"switch(variavel)": Avalia a variável escolhida a cada repetição do loop.

"case valor:": Define o bloco de código que será executado se a variável for igual ao "valor".

"break;": Ele interrompe a execução do switch (ou do for, dependendo de onde for colocado) e sai da estrutura. Se você esquecer o break no case, o Java continuará executando os próximos cases em sequência (efeito chamado fall-through).

"default:": O bloco "reserva". É executado se a variável não coincidir com nenhum dos case anteriores.
